# PlanIt Calendar

A simple and responsive wall calendar component built using React. This project is inspired by a physical wall calendar layout and focuses on clean UI, usability, and basic interactivity.

## Features

* Select a start and end date on the calendar
* Highlighted range between selected dates
* Month navigation (previous / next)
* Notes section to add and save personal notes
* Notes are stored using localStorage (no backend required)
* Dark mode / light mode toggle
* Fully responsive design (works on mobile and desktop)

## Design Approach

The idea was to create something that feels like a real wall calendar instead of a plain grid.
So I added:

* A top image to act as a visual header
* Clean spacing and simple colors
* A separate notes section for monthly planning

I also chose a student/coding-themed image to match the context of learning and productivity.

## Tech Stack

* React (Vite)
* JavaScript
* CSS (no external libraries)

## How to Run Locally

1. Clone the repository

   ```bash
   git clone https://github.com/MDakshayani/PlanIt-Calendar.git
   ```

2. Navigate to the project folder

   ```bash
   cd PlanIt-Calendar
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Run the development server

   ```bash
   npm run dev
   ```

5. Open in browser

   ```
   http://localhost:5174
   ```
